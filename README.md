# School Manager JDBC

Console application built with **Java + JDBC + MySQL** to manage students.

## Features
- CRUD Students
- Search by ID | name | email
- Export CSV

## Tech Stack
- Java
- JDBC
- MySQL
- Maven

## Database Setup

Before running the application, execute the SQL script:

`database.sql`

You can run it in MySQL Workbench or via terminal:

```sql
SOURCE database.sql;
